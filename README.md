# 批量重命名文件工具

## 📝 项目简介
这是一个用Python写的批量重命名工具，可以把当前文件夹下所有的 `.txt` 文件重命名为 `文件_1.txt`、`文件_2.txt`...

## ✨ 功能特点
- ✅ 自动查找当前目录下所有 `.txt` 文件
- ✅ 按文件名排序，保证结果稳定
- ✅ 避免覆盖已存在的文件
- ✅ 清晰的重命名日志
- ✅ 使用现代Python语法（pathlib）
